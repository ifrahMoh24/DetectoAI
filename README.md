# DetectoAI

DetectoAI is an AI-powered smartphone damage detection system.  
It combines a Flutter mobile app (`app/`) with a YOLO-based ML pipeline (`ml/`).

## Structure

- `app/` — Flutter mobile application (camera, UI, results)
- `ml/` — dataset, YOLO training, notebooks, scripts
- `docs/` — (optional) for report and diagrams
